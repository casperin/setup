# setup

A beginning point for creating very small es6 modules.

Contains:

* `.editorconfig`
* `eslintConfig`
  * `"modules": true`
  * `"browser": true,`
  * `"node": true,`
  * `"es6": true`
* `babel-core`
* `babel-preset-es2015`
* `babel-preset-stage-0`
* `browserify`
* `babelify`
* `watchify`

EditorConfig:

```js
root = true

[*]
indent_style = space
charset = utf-8
trim_trailing_whitespace = true
insert_final_newline = true
indent_size = 2
max_line_length = 80
```
